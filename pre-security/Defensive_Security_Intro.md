# Defensive Security Intro

## Objetivo
Comprender el rol de la seguridad defensiva y cómo se mitigan ataques comunes.

## Contexto
Se utilizó un escenario de un banco simulado donde se detectó actividad maliciosa
proveniente de una IP atacante.

## Acciones realizadas
- Identificación de la IP atacante
- Bloqueo de la IP sospechosa
- Configuración de límites de peticiones al servidor
- Creación de un plan de mitigación reutilizable

## Aprendizaje
Entendí que la seguridad defensiva se enfoca en prevenir, detectar y responder
a incidentes para proteger los sistemas y la información.

## Dudas
- Cómo se aplican estas medidas de forma automática en empresas reales.
