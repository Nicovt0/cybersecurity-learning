# What is Networking?
Conceptos básicos de redes vistos en TryHackMe:

- Una red permite la comunicación entre dispositivos.
- Los dispositivos pueden comunicarse dentro de una red.
- Internet conecta múltiples redes entre sí.
- El comando `ping` permite comprobar conectividad entre equipos.

Estos conceptos son la base para comprender cómo los sistemas
se comunican y cómo pueden producirse problemas de conectividad.
