# Blocking Malicious IP

## Situación
Se detectan múltiples peticiones sospechosas provenientes de una misma
dirección IP hacia un servicio web.

## Objetivo
Reducir el impacto del ataque y proteger la disponibilidad del servicio.

## Acciones
- Identificar la IP origen del tráfico sospechoso
- Bloquear la IP en el firewall o sistema de protección
- Limitar el número de peticiones permitidas
- Registrar el incidente para referencia futura

## Resultado esperado
Disminución del tráfico malicioso y continuidad del servicio.
